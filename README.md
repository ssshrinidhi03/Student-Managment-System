Student Management System

Overview

The Student Management System is a Python console application that allows users to manage student records efficiently. The program stores student information such as name, roll number, subject marks, average marks, and grade.

This project was created to practice Python fundamentals including:
- Functions
- Lists
- Dictionaries
- Conditional Statements
- Loops
- User Input and Output

Features

1. Add Student

- Add a new student.
- Enter marks for:
  - Maths
  - Physics
  - Chemistry
  - English
  - Computer Science
- Automatically calculates:
  - Total Marks
  - Average Marks
  - Grade

2. Display Students

- View all students currently stored in the system.
- Shows:
  - Name
  - Roll Number
  - Average Marks
  - Grade

3. Update Student

- Update marks of an existing student.
- Recalculates:
  - Average Marks
  - Grade

4. Search Student

- Search for a student using their roll number.

5. View Marksheet

- Displays complete marksheet including:
  - All subject marks
  - Average Marks
  - Grade

6. Delete Student

- Remove a student record from the system using roll number.
